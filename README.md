# ChatGPT Style Search Engine

## Overview
This project is an AI-powered ChatGPT Style Search Engine developed using Python. It combines predefined logic, Wikipedia search, and the TinyLlama language model to provide intelligent and conversational responses to user queries.

## Features
- Current Date and Time Detection
- Indian Time Zone Support
- Wikipedia-Based Information Retrieval
- AI Response Generation using TinyLlama
- Interactive Command-Line Interface
- Natural Language Question Answering

## Technologies Used
- Python
- TinyLlama
- Hugging Face Transformers
- PyTorch
- Wikipedia API
- Google Colab

## Project Workflow

User Question
↓
Predefined Question Check
↓
Wikipedia Search
↓
TinyLlama AI Model
↓
Final Answer
